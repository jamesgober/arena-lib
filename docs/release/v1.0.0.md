# Release v1.0.0 ‚Äî Stable API

**Date:** 2026-05-21
**Compare:** [`v0.9.0...v1.0.0`](https://github.com/jamesgober/arena-lib/compare/v0.9.0...v1.0.0)

## Summary

`arena-lib` `1.0.0` is the **first stable release**. The 0.5 implementation milestone built the four primitives, the 0.9 audit milestone hardened and documented them, and this release tags that audited surface as the frozen 1.0 contract. There are **no code changes** since 0.9.0 ‚Äî the audit produced no further findings worth gating the release, so the 0.9 surface ships verbatim as 1.0.

The crate provides four collaborating allocator primitives behind a single safe-Rust surface:

- **`Arena<T>`** ‚Äî generational arena with stable [`Index`] handles. Slot recycling bumps a per-slot generation counter so stale handles are rejected without reference counting.
- **`Interner`** ‚Äî `hashbrown`-backed string interner that maps `&str` ‚Üî a compact 4-byte [`Symbol`] for O(1) intern / lookup / resolve.
- **`Bump`** ‚Äî multi-chunk bump arena. `alloc(&self, value)` returns `&mut T` at O(1), grows on demand, and resets in O(1) while retaining chunks for reuse. Destructors are not run on reset.
- **`DropArena<T>`** ‚Äî typed sibling of `Bump` that *does* run destructors, with the same alloc-from-`&self` ergonomics.

All four are tied together by a single `#[non_exhaustive]` `Error` enum and a `Result<T>` alias. The crate is `no_std`-compatible (`default-features = false`); `std` is the default for `std::error::Error` integration.

## Stability contract

Within the 1.x line, every change is **purely additive**:

- New methods on existing types.
- New variants on the `#[non_exhaustive]` `Error` enum.
- New types in new modules.

Anything that would break a 1.x caller ‚Äî removed items, renamed methods, signature changes, reshaped error variants ‚Äî is out of scope until a hypothetical 2.0.

The published surface is described in full in [`docs/API.md`](../API.md). The pre-1.0 audit findings (A1‚ÄìA4) and their resolutions are in [`docs/release/v0.9.0.md`](./v0.9.0.md).
